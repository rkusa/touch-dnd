Droppable

Example

Options

accept

Type
Selector or Function
Default
*

Controls which draggable elements are accepted by the droppable.

Multiple types supported:

Selector
A selector indicating which draggable elements are accepted.
Function
A function that will be called for each draggable on the page (passed as the first argument to the function). The function must return true if the draggable should be accepted.

Selector Example

{% highlight js %}$('#example-accept-selector').droppable({ accept: '.a' }){% endhighlight %}
a
b

Function Example

{% highlight js %}$('#example-accept-function').droppable({ accept: function(el) { return el.hasClass('b') } }){% endhighlight %}
a
b

activeClass

Type
String
Default
false

If specified, the class will be added to the droppable while an acceptable draggable is being dragged.

Example

{% highlight js %}$('#example-activeClass').droppable({ activeClass: 'active' }){% endhighlight %}

disabled

Type
Boolean
Default
false

Disables the draggable if set to true.

Example

{% highlight js %}$('#example-disabled').droppable({ disabled: true }){% endhighlight %}

hoverClass

Type
String
Default
false

If specified, the class will be added to the droppable while an acceptable draggable is being hovered over the droppable.

Example

{% highlight js %}$('#example-hoverClass').droppable({ hoverClass: 'drop-here' }){% endhighlight %}

scope

Type
String
Default
"default"

Used to group sets of draggable and droppable items, in addition to the accept option. A draggable with the same scope value as a droppable will be accepted.

Example

{% highlight js %}$('#example-scope').droppable({ scope: 'T' }){% endhighlight %}
T

Methods

destroy()

Removes the draggable functionality completely.

{% highlight js %}$('#example-destroy').droppable() $('#example-destroy').droppable('destroy'){% endhighlight %}

disable()

Disables the draggable.

{% highlight js %}$('#example-disable').droppable('disable'){% endhighlight %}

enable()

Enables the draggable.

{% highlight js %}$('#example-enable').droppable('enable'){% endhighlight %}

option()

Gets an object containing key/value pairs representing the current droppable options hash.

{% highlight js %}var options = $('#example-option').droppable('option'){% endhighlight %}

option(optionName)

Gets the value currently associated with the specified optionName.

{% highlight js %}var isDisabled = $('#example-option').droppable('option', 'disabled'){% endhighlight %}

option(optionName, value)

Sets the value of the droppable option associated with the specified optionName.

{% highlight js %}$('#example-option').droppable('option', 'disabled', true){% endhighlight %}

option(options)

Sets one or more options for the droppable.

{% highlight js %}$('#example-option').droppable('option', { disabled: true }){% endhighlight %}

Events

activate(event, ui)

Triggered when an accepted draggable starts dragging.

ui.item
the element being dragged

Example

{% highlight js %}$('#example-activate').droppable({ accept: '.c' }) .on('activate', function(e, ui) { alert(42) }){% endhighlight %}
c
d

create(event)

Triggered when the droppable is created.

deactivate(event, ui)

Triggered when an accepted draggable stops dragging.

ui.item
the element being dragged

Example

{% highlight js %}$('#example-deactivate').droppable({ accept: '.e' }) .on('deactivate', function(e, ui) { alert(42) }){% endhighlight %}
e
f

drop, dragenter, dragleave, dragover

These events are natively provided through the HTML5 Drag'n'Drop API.
